PART I

Brief — What & How简短报告：要做的事 与 怎么做

1.1 What we are building 要做什么

2025–2026 涌现了一批 memory-equipped multimodal agent（M3-Agent、WorldMM、MMA、MemVerse），让 agent 在长期交互中持续看视频、听音频、读对话，并把这些观察以 episode 形式写进 long-term memory。当用户提问时，agent 从 memory 里 retrieve K 条相关 episode，喂给 base VLM 生成答案。每条 episode 本身是多模 composition：视频帧 + caption + 音频转录 + scene metadata。
A new generation of memory-equipped multimodal agents (M3-Agent, WorldMM, MMA, MemVerse) continuously observe video, audio, and dialogue, writing those observations as episodes into long-term memory. At query time, the agent retrieves K relevant episodes and feeds them to a base VLM. Each episode is itself a multimodal composition: vision frames + caption + audio transcript + scene metadata.
当 agent 答错时，开发者想知道：是哪条 memory 里的哪种 modality 出了问题——是视频帧没看清？字幕写错了？还是音频转录出现了 hallucination？这三种失败模式需要完全不同的修复策略，但目前所有归因工具要么把整条 episode 当 atomic player（per-document Shapley），要么只在 single-shot classification 上做 flat per-modality 归因。没有人在 memory-equipped multimodal agent 上做过 (memory item × modality) 两层归因。
When such an agent answers incorrectly, developers need to know which memory item × which modality is responsible — a blurry frame, a mis-transcribed caption, an audio hallucination? Each failure mode demands a different fix. Yet existing attribution tools either treat each episode as an atomic player (per-document Shapley) or perform flat per-modality attribution on single-shot classification. No prior work performs joint (memory item × modality) two-level attribution on memory-equipped multimodal agents.
We bring Owen value (1977) — the classical hierarchical Shapley for grouped players — to the memory layer of multimodal agents, for the first time. 第一次把 Owen value 这一经典 cooperative game theory 工具落到 multimodal memory-equipped agent 的归因问题上。

1.2 Three contributions 三层贡献

01

Application novelty

First application of Owen value to multimodal memory-equipped agents. Deep novelty check (2026-05-11) confirms the four-way intersection — Owen × multimodal × memory agent × inference-time — is empty.

02

Closed-form allocation

Derive \(N_\text{out}^{*} \propto \sqrt{2^K}\) and \(N_\text{in}^{(k)\,*} \propto \sqrt{2^{L_k}}\) from the variance ratio bound. Given budget \(B\), the optimal split is closed-form — no hyperparameter tuning.

03

Modality-pruned retrieval

Attribution-driven eviction \(\tilde M(Q) = \{\, v_k^{(\ell)} : \phi_k > \tau_1 \,\wedge\, \psi_{k,\ell} > \tau_2 \,\}\). Target \(\ge 10\) pp \(\Delta\)Acc at \(\le 40\%\) context retention.

1.3 How — Method at a glance 方法一览

实现成本：~200 LoC patch 到 m3_agent/retrieval.py，0 training，frozen base model + frozen retrieval。M3-Agent 的 memory 本身是 entity-centric multimodal graph（每个 entity 下挂 modality sub-node），(item × modality) 两层结构是已有数据结构，我们不是 patch 一个 hierarchy 进去，是激活已有的。
Implementation cost: ~200 LoC patch on m3_agent/retrieval.py; no training; frozen base model and frozen retrieval. M3-Agent's memory is already an entity-centric multimodal graph with modality sub-nodes — the two-level structure is the native data structure, not a hierarchy we impose.

1.4 Why this setting fits Owen value 为什么这个 setting 对 Owen 值特别合适

Shapley 与 Owen value 要求 permutation invariance——必须能对任意子集评估 u(S)。RL credit (如 OSPO) 受困于 trajectory 的因果时序，无法 freely mask 中间 action。但 retrieval 把 agent 的动态因果流切成了静态 snapshot 集合——K 条 retrieved item 在 generator context 里并行 attend，可以自由 mask 任意子集。memory 的真正时序在 write-time，那一层确实 attribution-hostile——我们主动切割那层，留给 backup direction (Cross-Modal Causal Mediation for Write-Policy) 处理。
Shapley/Owen require permutation invariance: u(S) must be defined for any subset S. RL credit (e.g., OSPO) is constrained by trajectory causality — you cannot freely mask an interior action. Retrieval, however, turns a dynamic causal stream into a static snapshot set: the K retrieved items attend in parallel and any subset can be masked. The genuine temporal structure lives in write-time consolidation; we deliberately scope it out as a non-goal and reserve it for the backup direction (Cross-Modal Causal Mediation for Write-Policy).

PART II

Theory — Owen Value, Variance, Allocation理论：Owen 值 · 方差界 · 采样配置

2.1 Background: Owen value (1977) 背景：Owen 值

Owen value 是 cooperative game theory 中处理 "玩家有先验分组（a priori union）" 情境的经典扩展。把玩家 N 划分为 coalitions {P_1, ..., P_K}：先在 P_k 之间计算 between-group Shapley（φ_k），再在每个 P_k 内部计算 within-group Shapley（ψ_{k,ℓ}）。Kamijo (2009) 给出了等价的 "Two-Step Shapley" 表述。
Owen value (Owen 1977) is the classical Shapley extension for games whose players have an a priori coalition structure. Partition the player set N into coalitions {P_1, ..., P_K}: compute a between-group Shapley value φk first, then for each Pk compute a within-group Shapley value ψk,ℓ. Kamijo (2009) gives the equivalent "Two-Step Shapley" formulation.
Conservation (Owen Efficiency) \[ \begin{aligned} \text{between-group:}\quad & \sum_{k=1}^{K} \phi_k \;=\; U(M) - U(\emptyset) \\[4pt] \text{within-group:}\quad & \sum_{\ell=1}^{L_k} \psi_{k,\ell} \;=\; \phi_k \qquad \forall\, k \end{aligned} \]
Honest framing

Owen value is classical. We do not claim a new theorem on hierarchical Shapley axioms. The novelty lies in (a) applying it to multimodal memory agents for the first time, (b) deriving a closed-form sample-allocation policy from its variance ratio bound, and (c) using it to power modality-pruned retrieval.

2.2 Our setup 我们的 setup

Player set (query-conditioned) \[ M(Q) \;=\; \{\, m_1, \ldots, m_K \,\}, \qquad m_k \;=\; \bigl( v_k^{(1)}, \ldots, v_k^{(L_k)} \bigr) \] Utility function \[ u(y, y^*) \;=\; \mathbf{1}[y = y^*] \;\in\; \{0, 1\}, \qquad U(S) \;=\; \mathbb{E}_y\!\bigl[\, u(y, y^*) \,\big|\, Q, S \,\bigr] \;\in\; [0, 1] \] Two games \[ \begin{aligned} G_\text{out}: \quad & \text{players} = \{m_1, \ldots, m_K\}, \quad \text{utility} = U \\ G_\text{in}^{(k)}: \quad & \text{players} = \text{modalities of } m_k, \quad \text{utility} = U(\,\cdot\,\mid M_{\neg k} \text{ frozen}) \end{aligned} \]

2.3 Proposition 2 — Finite-sample variance 命题 2：有限样本方差

对每层 Monte Carlo 估计器，使用 N 个采样：方差被 sub-coalition lattice 大小线性 bound 住。Utility u ∈ {0,1} 给出 McDiarmid bounded-difference 常数为 1，σ²_max ≤ 1/4。
For the Monte Carlo estimator at each level with N samples, the variance is linearly bounded by the sub-coalition lattice size. Since u ∈ {0,1}, the McDiarmid bounded-difference constant is 1 and σ²max ≤ 1/4.
(2a) Variance bound \[ \operatorname{Var}(\hat\phi_k) \;\le\; \frac{2^K}{4N}, \qquad \operatorname{Var}(\hat\psi_{k,\ell}) \;\le\; \frac{2^{L_k}}{4N} \] (2b) McDiarmid concentration \[ \mathbb{P}\!\Bigl( \bigl| \hat\phi_k - \phi_k \bigr| > \varepsilon \Bigr) \;\le\; 2 \exp\!\Bigl( - \frac{2 N \varepsilon^2}{2^K} \Bigr) \] (2c) Variance ratio vs flat token Shapley \[ \frac{\operatorname{Var}(\text{hierarchical})}{\operatorname{Var}(\text{flat})} \;\le\; \frac{\sum_{k=1}^{K} 2^{L_k}}{2^{\sum_{k=1}^{K} L_k}} \]

2.4 Algorithm 1 — Closed-form allocation 算法 1：闭式采样配置

这是 v2 相对原 plan 的新贡献。给定总预算 B = K · N_outer + Σ_k N_inner^(k)，对 (2a) 的 variance 求和并施加预算约束，用拉格朗日乘子法解最优配置：
This is the new algorithmic contribution beyond Owen 1977. Given a total query budget B = K · Nouter + Σk Ninner(k), minimise the total Var via Lagrangian optimisation:
Lagrangian objective \[ \min_{N_\text{out},\, \{N_\text{in}^{(k)}\}}\;\; \frac{2^K}{N_\text{out}} \;+\; \sum_{k=1}^{K} \frac{2^{L_k}}{N_\text{in}^{(k)}} \quad\text{s.t.}\quad K \cdot N_\text{out} \;+\; \sum_{k=1}^{K} N_\text{in}^{(k)} \;=\; B \] Closed-form optimal allocation \[ N_\text{out}^{*} \;\propto\; \sqrt{2^K}, \qquad N_\text{in}^{(k)\,*} \;\propto\; \sqrt{2^{L_k}} \] Practitioner interface \[ \bigl(K,\; \{L_k\},\; B\bigr) \;\;\longmapsto\;\; \bigl(N_\text{out}^{*},\; \{N_\text{in}^{(k)\,*}\}\bigr) \]

2.6 Phase 4 theory upgrade — matching lower bound Phase 4 理论加强：匹配下界

为把 Prop 2 从 "我们有一个 upper bound" 升级为 "我们有 the bound"，Phase 4 计划补一个 information-theoretic lower bound：任何 unbiased estimator 在 hierarchical regime 都不能比 Ω(2^K/N + Σ 2^{L_k}/N) 更好。技术路径借鉴 Mann-Wright (1991) 经典 Shapley estimator lower bound，把 single-coalition 推广到 nested coalition。
To upgrade Prop 2 from "we have a bound" to "we have the bound", Phase 4 adds an information-theoretic lower bound: any unbiased estimator in the hierarchical regime must satisfy Var ≥ Ω(2K/N + Σ 2L_k/N). We follow Mann–Wright (1991) for the single-coalition argument and extend to nested coalitions.

2.7 What we deliberately do NOT claim 明确不卖的几件事

  • 不卖 "two-level Shapley factorization" 为新数学结构（= Owen 1977）
  • 不卖 Conservation 为新定理（= Owen Efficiency axiom）
  • 不卖 Uniqueness on hierarchical lattice 为新结果（= Owen characterization）
  • 新声明只有：(a) 应用 novelty, (b) closed-form allocation, (c) modality-pruned retrieval 实证
